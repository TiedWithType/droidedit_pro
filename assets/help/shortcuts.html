Shortcuts

Configuring and using shortcuts (even without a physical keyboard).

Summary

  • Default Shortcut List
  • Changing Shortcuts
  • Soft Keyboard Support

Default Shortcut List

There are several predefined keyboard shortcuts you can use with DroidEdit. You can check the complete list by opening the Keyboard Shortcuts preference.

Changing Shortcuts

To change a shortcut, go to the Keyboard Shortcuts preference and press the shortcut you want to change. You'll then be able to choose a new shortcut by pressing the desired key combination.

Soft Keyboard Support

Those who don't have a physical keyboard can still use shortcuts with DroidEdit. To use the volume up physical key as Ctrl just activate the Keyboard Shortcuts → Volume Up as Ctrl preference.

You can also use volume down as Tab by activating the Keyboard Shortcuts → Volume Up as Tab preference.

To use shortcuts with a virtual keyboard you will also have to activate the Keyboard Shortcuts → Soft Keybaord Support preference.
