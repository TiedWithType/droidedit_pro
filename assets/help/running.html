DroidEdit Help

Running

DroidEdit is a code editor, not an IDE. So there is no compile and run features included. However, there are a few ways you can run your applications directly from DroidEdit.

Summary

  • Using SL4A
  • Using an SFTP server
  • Using a local command

Using SL4A

SL4A (Scripting Layer for Android) brings scripting languages to Android by allowing you to execute scripts directly on the Android device. These scripts have access to many of the APIs available to full-fledged Android applications, but with a greatly simplified interface that make it easy to get things done.

To execute a file written in one of the supported languages (Python, Perl, JRuby, Lua, BeanShell, JavaScript, Tcl, and shell) just use the menu item File Actions → Execute Using SL4A. You can either execute in the background or in a terminal window.

Don't forget to install SL4A first. You also need to install the module for the language you want to use.

Using an SFTP server

If you're editing a file that resides in an SFTP server (see the Remote Files section), you can execute the file directly on the server.

To do this you have to create a new external command using the External Commands → Add External Command preference.

An external command has a name (that is used to identify it when picking a command from a list), the command per se and a result action (how to show the result of the command).

When writing the command you can use a few predefined placeholders: ${file} for the current filename, ${name} for the filename but without the extension and ${path} for the file path. You can also define your own placeholders, using the same format, that will be asked at runtime.

The result action can be:

  • Do nothing: self explanatory
  • Show results: in a dialog
  • Open in a new document: useful if you want to do something with the output of the command
  • Replace current document: useful if the command is some kind of transformation filter

To run an external command, you just have to use the File Actions → Run External Command menu item when editing an SFTP file.

Using a local command

External commands can also run locally. This feature is most useful if your phone is rooted and you have busybox installed.

To run an external command locally, you just have to use the File Actions → Run External Command menu item when editing a local file.
