File Management

How to open, save and create new files.

Summary

  • Create a new File
  • Open an existing File
  • Save a File
  • Close a File

Create a new File

To create a new file, you can either use the menu item New or use the shortcut Ctrl+N.

If you're using the old menu configuration you can also long press the New file menu icon to create a new file using a different encoding than the default one.

Open an existing File

To open an existing file, you can either use the menu item Open or use the shortcut Ctrl+O.

You can also use the left side navigation drawer by sliding from the left edge of the screen.

You will be presented with a menu that will allow you to choose the file system from which you would like to open the file. Currently, you can choose between opening locally, on a Dropbox account, a Box account or in any one of the configured SFTP, FTP and FTPS accounts.

To learn how to configure this accounts, check the Remote Files section of this manual.

After selecting the file system, you'll be able to navigate through the file system's tree and choose the file you wish to open.

If you're using the old menu configuration you can also long press the Open file menu icon to open an existing file using a different encoding than the default one.

Save a File

To save an opened file, you can either use the menu item Save or use the shortcut Ctrl+S.

If the file hasn't been saved yet, you will be presented with a menu that will allow you to choose the file system in which you would like to save the file. Currently, you can choose between saving locally, on a Dropbox account, a Box account or in any one of the configured SFTP, FTP and FTPS accounts.

To learn how to configure this accounts, check the Remote Files section of this manual.

After selecting the file system, you'll be able to navigate through the file system's tree and choose the location and filename of the new file. You will also be able to create new directory by pressing the New Folder icon in the top right corner of the file explorer.

If you want to save the file to a different location, you can either use the menu item Save as... or use the shortcut Ctrl+Shift+S.

Close a File

To close an opened file, you can either use the menu item Close or use the shortcut Ctrl+W.

If the file has been modified since it was last saved, you'll be presented with a dialog asking if you are sure about closing the file.
