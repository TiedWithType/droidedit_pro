Appearance

Changing the overall appearance of DroidEdit.

Summary

  • Font Size
  • Line Numbers
  • Color Themes
  • Ui Layout

Font Size

You can change the font size by editing the Appearance → Font Size preference.

Line Numbers

If you want to show the file line numbers in the left side of the screen, just turn on the Editor → Line Numbers preference.

Color Themes

DroidEdit already comes with 20 pre-installed color themes. Besides the background and foregorund color used for text they also contain several other colors that are used for syntax highlighting.

To change theme, just go to Appearance → Theme and choose another theme.

You can also create your own themes by going to Appearance → Manage Custom Themes → Create Theme and choose another theme. After creating a new theme it will be available in the list of themes for you to choose.

When creating or editing a theme, you'll be presented with a list of different types of tokens. To change the color used for each one of these tokens just click it and choose a new color using a color picker.
