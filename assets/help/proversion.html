Pro Version Features

The pro version of DroidEdit has the following extra features:

  • Remote Files: Dropbox, Box, SFTP, FTP, FTPS and FTPES
  • Root mode: To open system files
  • External commands: To open system files
  • Emmet: Previously Zen Coding
  • No Ads
